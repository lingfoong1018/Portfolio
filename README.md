## UNSW Degree Planner (Team Project)

GitHub: https://github.com/unsw-cse-comp99-3900/capstone-project-25t2-3900-t09a-date

This project was developed as part of a team of 6.

We built a full-stack degree planning web app that helps students track program requirements and manage course selections, including an admin interface for editing complex program structures through a user-friendly UI.

Tech Stack: React, JavaScript, Flask, PostgreSQL, REST APIs

Demo: https://youtu.be/_Mhw5aTSa6k

### My Contributions (Frontend Developer)
- Designed and implemented an admin interface for editing complex degree program structures stored as nested JSON
- Built React-based forms and UI components to allow non-technical users to easily modify structured data
- Implemented CRUD functionality for program data, including adding/removing sections, fields, and course lists
- Integrated local storage and session handling to simulate authentication and persist user state
- Collaborated in a team using Git workflows, contributing to feature development and debugging

### What I learned
- The importance of validation and data integrity in system design
- The importance of preserving immutability for deeply nested states for reliable UI rendering
- Mapping complex backend data structures (JSON) to intuitive frontend interfaces for non-technical users
- Designing CRUD interfaces that ensures both flexibility and usability
- Debugging issues across frontend and backend, including data formatting and API interactions
- Working in a team using Git workflows, feature branches, and iterative development with evolving requirements
